# BotGame20: Telegram RPG Battle Bot

Проект интерактивного Telegram-бота для проведения пошаговых сражений. Реализован на языке Python с использованием библиотеки \`aiogram\`.

## Технологии
* **Язык:** Python 3.x
* **Библиотека:** aiogram 2.x
* **Архитектура:** Обработка событий в реальном времени, использование Inline Keyboards для управления.

## Основной функционал
* **Боевая механика:** Реализована логика пошагового боя (атака, защита, расчет урона).
* **Интерактивность:** Игровое меню и выбор действий через встроенные кнопки Telegram.
* **Масштабируемость:** Код подготовлен для внедрения системы классов персонажей и прокачки уровней.

## Запуск проекта

1. **Клонирование репозитория:**
   - git clone https://github.com
   - cd BotGame20

2. **Установка зависимостей:**
   pip install aiogram==2.25.1   

3. **Настройка:**
   * Получите токен у [@BotFather](https://t.me).
   * Вставьте ваш токен в переменную *TOKEN* в файле *.env*

4. **Запуск:**
   \`
   python main.py
   \`
